# infocard-video-skills

Open-source skills for information cards and Remotion video workflows.

## Included now

- `skills/editorial-card-screenshot`: generate editorial-style HTML info cards and capture them as ratio-specific screenshots.

## Repository layout

- `skills/`: reusable skills
- `docs/`: supporting prompts and documentation

## Quick start

From the repository root:

```bash
./skills/editorial-card-screenshot/scripts/capture_card.sh input.html output.png 3:4
```
